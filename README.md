# Repeated Speech

Convert text to sound and speak repeatedly after an interval

## Getting Started

### Prerequisites

##### Linux

```
sudo apt install espeak
```
##### Windows

### Installing

##### Linux

```
npm install
npm run build // run webpack
npm run dist // create distribution package
```
##### Windows
